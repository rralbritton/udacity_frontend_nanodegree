About

Summary

Selected Projects

Network Team

Javascript, HTML, CSS

Worked closely with the Network Engineer to create a mapping solution that would be eye-catching and easy to interpret for members at all levels of their organization. The end-result was a custom built application using ArcGIS Calcite Maps framework and ESRI's Javascript 4.x API.

Shuttle Tracker, Campus Map

Javascript, HTML, CSS

Integrated a third party application to track live shuttle locations within the University of Utahs Campus Map. Other contributions to the campus map includes the overall maintenace and updates of frontend code (Javascript, HTML, CSS) which is built on ESRI Javascript 3.x API.

Walk After Dark

Javascript, HTML, CSS

Created a custom, mobile, crowdsourcing solution for faculty, staff, and students to report on lighting and safety issues around campus

Oceanfront Viewshed Analysis

Python, Custom Tool Development

Built a custom GIS tool that could be used to run a viewshed analysis on thousands of sites for both pre-construction and post-construction timeframes. The output of the tool provided information on each parcels viewshed in degrees, allowing for further evaluation of the effects of dune construction on New Jersey shore residents oceanfront views.

SBR Potential Core Forest

Suitability Modeling, Model Builder, Python

The objective of this analysis was to delineate and describe potential core forest within the Southern Blue Ridge ecoregion for the purpose of informing conservation actions. Results were used to conduct a compatibility assessment for Nantahala-Pisgah National Forest Land and Resource Management Plan revision.
